CSS

  • A margin indicates how much space we want around the outside of an element.
  • Inline CSS: style directly in html file, using a style attribute with CSS rule to style a particular page element ONLY. One bad thing however is, each inline stlye must be edited INDIVUALLY whenever change is needed.
  • Intertnal style sheet: contains css rules in the head section of the html file.
  • External CSS style sheet: MOST CONMMON, keep css rules in seperate file. example of linking it is in this file.
  • A selector defines the elements or attributes to which the rules, or declartions will apply.
  • Declations contain two parts, propety/value. The property that we would like to apply and the value to that propety.
  • margin indciates how much space we would like around the OUTSIDE of the document, setting it to 0 leavrs no extra space.
  • padding adds space around the content on the INSIDE of the element, when increasing padding you are making an entire element bigger.
  • "*" is a special wild card symbol which allows us to apply rule to all visible elements on the page.
  • Some selector examples are, h1, header, footer, img, ul, etc...
  • You can combine two selectors which helps us write "Do not repeat yourself"/DRY code.
  • To assign a class in css you use a period before naming the class you are assigning. For example, in this files css document, the period before card is designating this as a class slector.
  • A padding indicates how much space we want around the content inside an element.
